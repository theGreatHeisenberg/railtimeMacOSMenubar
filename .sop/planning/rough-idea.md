# Rough Idea

Create a Caltrain scheduler for MacOS - preferably a widget or menu bar app that allows quick viewing of next trains/schedules.

## Existing Resources
- Webapp deployed at: https://railtime.pages.dev/
- GitHub repository: github.com/theGreatHeisenberg/railtime
- APIs from the webapp can be reused

## Key Requirements
- Quick access from MacOS (top nav bar or widget)
- View next trains/schedules at a glance
- Leverage existing railtime APIs
